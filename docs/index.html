Crypto Lab · 密码学实验室

从零开始用 Python 跑通密码学核心概念

一套面向初学者的密码学动手练习。不追求数学证明，先用代码建立直觉，再回头补理论。

包含内容

  • 古典密码：凯撒密码、频率分析
  • 对称加密：AES-256-GCM
  • 哈希与认证：SHA-256、HMAC、Argon2id
  • 非对称加密：RSA 加密与数字签名
  • 真实世界协议：TLS/HTTPS 证书检查
  • 进阶主题：Diffie-Hellman、ECDSA、Fernet、JWT

快速开始

git clone https://github.com/vajhXajhcv/crypto-lab.git
cd crypto-lab
python -m venv venv
source venv/bin/activate  # Windows: .\venv\Scripts\activate
pip install -r requirements.txt
pytest -v
