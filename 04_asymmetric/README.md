# 04 非对称加密

## 目的

理解公钥密码学：RSA 加密、解密、签名、验证。

## 文件

- `rsa_basic.py`：RSA 完整演示

## 运行

```bash
cd crypto-lab/04_asymmetric
python rsa_basic.py
```

## 关键概念

- **公钥加密，私钥解密**：保护机密性
- **私钥签名，公钥验证**：保护完整性和不可否认性
- **混合加密**：实际用 RSA 加密 AES 密钥，再用 AES 加密大数据

## 注意

- RSA 密钥至少 2048 位（推荐 3072 或 4096）
- ECDSA/Ed25519 通常比 RSA 更快、密钥更短
